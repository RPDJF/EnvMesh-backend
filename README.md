# EnvMesh Backend

EnvMesh Backend is the server-side component of the EnvMesh project.  
It provides the core APIs, services, and infrastructure logic required to
operate EnvMesh within a single organization.

This repository is intended for **self-hosted and internal organizational use**.

---

## Project Intent

EnvMesh is developed with the long-term goal of supporting a hosted offering.
At present, this repository is released exclusively for personal, educational,
and internal organizational use under the EnvMesh Source-Available License.
